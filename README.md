# ✍️ Handwritten Digit Recognizer

A Machine Learning + Python project that recognizes handwritten digits using a CNN model and a GUI.

## 🚀 Features
- Draw digits using mouse
- Real-time prediction
- Confidence score display
- Clean GUI using Tkinter

## 🧠 Tech Stack
- Python
- TensorFlow
- Tkinter
- PIL (Pillow)

## ▶️ How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Train model
python train_model.py

### 3. Run app
python app.py

## 📸 Output
Draw a digit → Click Predict → Get result instantly
